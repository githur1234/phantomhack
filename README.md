# PhantomHack 🐍


## 📌 Hakkında

`PhantomHack`, Flask tabanlı bir uzaktan sistem kontrol arayüzüdür. Bu araç, bir hedef sistem hakkında bilgi toplamak, ekran görüntüsü almak, kamera yayını izlemek, komut çalıştırmak gibi işlevleri barındırır. Ayrıca istenirse temel bir keylogger modülüyle de genişletilebilir.

## 🚀 Özellikler

- 📷 Canlı kamera yayını (VideoCapture)
- 🖥️ Ekran görüntüsü yayını
- 🔍 Sistem bilgileri (CPU, RAM, disk, IP, MAC, batarya, çalışan işlemler)
- 🖱️ PyAutoGUI ile uzaktan ekran görüntüsü alma
- ⚙️ Komut çalıştırma formu (Terminal)
- 🔑 Keylogger (deneysel)
- ⏻ Uzaktan yeniden başlatma ve kapatma komutları
